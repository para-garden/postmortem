# postmortem

earth, but no people.

a worldbuilding project rendered as a spatial graph. places, things, machines, silences — positioned because of how they relate, not listed because of when they were written.

no decay. no animals. nothing that moves of its own volition. the buildings still stand, the metro still arrives, the wind still blows through the open window.

built on the [ptera.world](https://ptera.world) graph engine.
